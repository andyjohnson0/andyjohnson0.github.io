PRIVACY POLICY MODEL FOR MOBILE APPLICATIONS

This privacy policy governs your use of the software application Plain Text (“Application”) for mobile devices that was created by Andrew Johnson. The Application is used to remove formatting from text on the clipboard.

This is version 1 of this privacy policy. A version history of any changes is provided below.

What information does the Application obtain and how is it used?

The Application modifies text stored in the device clipboard. It does not obtain or modify any other information. The Application does not have permission to access the Internet and cannot communicate with any external device or service.

User Provided Information

The Application does not require the user to provide any information. The Application does not require the user to register.

Automatically Collected Information

The Application does not automatically collect any information.

Does the Application collect information about the location of the device?

No.

Do third parties see and/or have access to information obtained by the Application?

No.

What are my opt-out rights?

To opt-out of using the Application, uninstall it using the Android Settings app. There is nothing else to opt-out of.

Data Retention Policy

No data is retained by the Application.

Children

The Application is suitable for use by children.

Security

The Application does not store information on the device, except transiently on the device clipboard. The Application does not access the Internet in any way.
